# anthony-de.github.io

Basic TypeScript project configured with `pnpm` and Vite for GitHub Pages.

## Setup

```bash
pnpm install
```

## Local development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy to GitHub Pages

```bash
pnpm deploy
```

This deploy script publishes the `dist/` folder to the `gh-pages` branch.
